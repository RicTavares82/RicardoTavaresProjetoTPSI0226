5/2016 - Decisão do projeto para apresentação.
    Este trabalho será sobre a construção de uma app que irá receber valores de clientes e irá servir para apresentar estatísticas sobre os clientes, como por exemplo, a média de idade dos clientes, a quantidade de clientes por cidade, entre outras informações relevantes. A app será desenvolvida utilizando a linguagem de programação Python. O objetivo é fornecer uma ferramenta fácil de usar para que os usuários possam visualizar e analisar os dados dos clientes de forma eficiente. O sistema vai permitir o registo, validação, armazenamento e manipulação de dados de forma persistente.
